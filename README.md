# Шашки с Ромашкой — как получить APK через GitHub

Тебе **ничего не нужно устанавливать на компьютер**. Сборка происходит на серверах GitHub,
ты только нажимаешь кнопки в браузере. Весь путь — минут 15 в первый раз.

---

## Что это за файлы

Это готовый проект приложения. Внутри:
- `app/src/main/assets/index.html` — само приложение (то, что ты тестировал).
- `.github/workflows/build.yml` — инструкция, по которой GitHub сам соберёт `.apk`.
- остальное — техническая обвязка Android, трогать не нужно.

Если захочешь потом поменять задания или цвета — правится только `index.html`,
заливаешь заново, и GitHub собирает новый `.apk`.

---

## Шаг 1. Аккаунт GitHub (один раз)

1. Зайди на **github.com** → кнопка **Sign up**.
2. Введи почту, пароль, имя пользователя. Подтверди почту. Это бесплатно.

## Шаг 2. Создать репозиторий (папку проекта на GitHub)

1. Вверху справа нажми **+** → **New repository**.
2. **Repository name**: например `shashki-romashka`.
3. Оставь **Public**. Ничего больше не отмечай.
4. Нажми **Create repository**.

## Шаг 3. Залить файлы проекта

1. Распакуй мой архив `ShashkiRomashka.zip` у себя на компьютере.
2. На странице нового репозитория найди ссылку **uploading an existing file**
   (или кнопка **Add file** → **Upload files**).
3. Открой распакованную папку, **выдели всё содержимое** и перетащи в окно браузера.
   - Тащи именно то, что **внутри** папки `ShashkiRomashka` (файлы `build.gradle`,
     `settings.gradle`, папки `app` и `.github`), а не саму папку.
   - ⚠️ Папка **`.github`** обязательна — в ней лежит инструкция сборки.
     На Mac скрытые папки показываются по **Cmd + Shift + точка**.
     На Windows она видна обычным образом.
4. Внизу нажми зелёную кнопку **Commit changes**.

## Шаг 4. Дождаться сборки

1. Вверху репозитория открой вкладку **Actions**.
2. Увидишь запись **Build APK** с жёлтым кружком (идёт сборка).
   Через 3–5 минут он станет **зелёной галочкой** — готово.
   - Если красный крестик — открой запись, я помогу разобраться по тексту ошибки.

## Шаг 5. Скачать APK

1. Кликни на завершённую (зелёную) сборку.
2. Прокрути вниз до раздела **Artifacts**.
3. Нажми **shashki-romashka-apk** — скачается небольшой zip.
4. Распакуй его — внутри файл **`app-debug.apk`**. Это и есть приложение.

## Шаг 6. Установить на Samsung Z Fold 5

1. Перекинь `app-debug.apk` на телефон (по кабелю, через облако или отправь себе в мессенджер).
2. Открой файл на телефоне (через «Мои файлы»).
3. Телефон попросит разрешить **установку из неизвестных источников** для этого приложения —
   разреши.
4. Нажми **Установить**. Готово — иконка «Шашки с Ромашкой» появится в меню.

## Частые вопросы

**Вкладка Actions пустая / сборка не началась.**
Скорее всего не залилась папка `.github`. Создай файл вручную:
на странице репозитория **Add file** → **Create new file**, в имени впиши
`.github/workflows/build.yml` (косые черты сами создадут папки), вставь содержимое
из файла в архиве по тому же пути, **Commit**.

**Безопасно ли это?** APK подписан тестовым (debug) ключом — для установки себе на телефон
это нормально и работает без проблем. Для публикации в Google Play он не предназначен —
если когда-нибудь захочешь в Play, понадобится отдельная (релизная) подпись, поможем тогда.

**Как обновить приложение?** Поменяй `index.html` в репозитории
(**Add file** → перезалить, или открой файл → карандаш → правка), **Commit** —
сборка запустится снова, скачаешь новый `app-debug.apk` из Actions.
